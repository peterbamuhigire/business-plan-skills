# Budgeting and Forecasting Best Practices

*Sources: Donnelly & Foley, Budgeting for Better Performance, 4th ed. (ILM/Elsevier/Pergamon, 2003); Morlidge & Player, Future Ready: How to Master Business Forecasting (Wiley, 2010); Vandeput, Demand Forecasting Best Practices (Manning, 2023).*

---

## 1. Budget Foundations (ILM)

### What a Budget Is

A budget is a **quantitative plan of action prepared in advance of a defined period of time**. Four essential features:
- **Quantitative** — expressed in figures (money, units, hours)
- **Prepared in advance** — drawn up before the period it covers
- **Period-specific** — relates to a defined timeframe (usually one year)
- **Plan of action** — guides decisions, not a rigid constraint

### Common Types of Budget

| Budget Type | Purpose | When It's the Key Budget |
|---|---|---|
| **Sales budget** | Forecast of revenue and volume | Most common limiting factor in established businesses |
| **Cash budget** | Timing of cash in/out; bank balance | Most common limiting factor for new/small businesses |
| **Production budget** | Output volumes and manufacturing costs | When capacity, not demand, is the binding constraint |
| **Departmental costs** | Operating expenditure by function | Derived from the key budget |
| **Capital expenditure budget** | Investment in assets | Driven by growth/operational requirements |

### The Limiting (Key) Budget Factor

The **key budget factor** is the constraint that limits all other budgets — it must be produced first:

- **New business / cash-constrained:** Cash budget is the key budget. The bank manager will insist on it before authorising any loan. Show monthly opening balance, inflows, outflows, and closing balance.
- **Established commercial business:** Sales budget is the key budget. Everything else (purchasing, staffing, production) follows from what you plan to sell.
- **Capacity-constrained business:** Production budget is the key budget (when you can sell more than you can make).

> **Business plan implication:** Always identify the limiting budget factor first. A new Uganda SME starting with limited capital should build the cash budget before the income statement — the business lives or dies by its cash position, not its accounting profit.

### Purposes of Budgeting

1. **Planning** — forces formal thinking about the future rather than reacting to events
2. **Control** — provides a yardstick to measure actual performance against
3. **Co-ordination** — aligns departments toward common goals (a 10% sales increase is pointless if production plans a 5% cutback)
4. **Communication** — gets management to discuss policies and targets
5. **Motivation** — involvement in setting budgets increases ownership

**Critical caution:** Budgets should never be so inflexible as to prevent sensible decisions. The budget is a plan, not a rule. If a competitor launches a new product, the right response is to adapt the budget, not hide behind it.

## 2. Budgetary Control — Variance Analysis

### The Control Cycle

```
1. Set budget (plan)
        ↓
2. Measure actual results
        ↓
3. Identify variances (actual - budget)
        ↓
4. Assign responsibility
        ↓
5. Investigate causes
        ↓
6. Take corrective action (or revise budget if circumstances have changed)
```

### Reading Variances

| Variance Type | Sales | Costs |
|---|---|---|
| **Favourable (F)** | Actual > Budget | Actual < Budget |
| **Adverse (A)** | Actual < Budget | Actual > Budget |

**Example budgetary control report:**

| Line Item | Budget | Actual | Variance |
|---|---|---|---|
| Sales revenue | 600,000 | 700,000 | 100,000 F |
| Materials | 250,000 | 280,000 | 30,000 A |
| Wages | 100,000 | 120,000 | 20,000 A |
| Admin salaries | 55,000 | 50,000 | 5,000 F |
| Advertising | 15,000 | 20,000 | 5,000 A |
| **Operating profit** | **115,000** | **165,000** | **50,000 F** |

Even a favourable variance should be investigated — short cuts may have been taken, or lessons can be learned for future budgets.

### Controllable vs. Non-Controllable Costs

- **Controllable:** costs the manager can directly influence (staffing levels, material purchases, marketing spend)
- **Non-controllable:** costs charged to a department but outside the manager's authority (building depreciation, allocated central overheads, interest on group borrowings)

Budget reports should distinguish these to ensure fair accountability.

### Flexible Budgets

A **fixed budget** assumes one level of output. A **flexible budget** adjusts for actual activity level, separating the volume effect from the efficiency effect.

**Why this matters:** If you budget for 1,000 units and sell 1,200, your costs will naturally be higher — but that's not a problem, it's a consequence of higher sales. A fixed budget would flag an adverse cost variance; a flexible budget corrects for volume and shows whether you are actually spending more or less efficiently per unit.

**Flexible budget formula:**
```
Flexible Budget Cost = (Fixed Cost Component) + (Variable Cost per Unit × Actual Units)
```

Set up your Year 1 monthly budget with fixed and variable costs clearly separated so that monthly reviews compare "what should we have spent at this actual volume" versus "what did we spend."

### Standard Costing

Standard costing goes one level deeper than budgeting — it sets a **predetermined cost per unit** rather than just a total department budget. Useful for manufacturing and food production businesses where unit cost control matters:

- **Standard cost** = what one unit should cost under normal operating conditions
- **Variance analysis** identifies whether variance came from volume, price (cost per input), or efficiency (inputs used per unit)

## 3. Performance Measurement Framework (ILM)

### Principles of Good Performance Measurement

Performance measurement considers how well something performs compared with:
1. Its **own past performance** (trend measurement)
2. A **predetermined target** (budget/plan vs. actual)
3. **External comparisons** (competitors, benchmarks, industry norms)

Always make clear which comparison basis is being used — the same performance can look excellent or poor depending on the reference point.

### Financial Performance Measures

| Measure | Formula | Signal |
|---|---|---|
| Gross margin % | Gross profit ÷ Revenue × 100 | Pricing power and COGS efficiency |
| Net margin % | Net profit ÷ Revenue × 100 | Overall profitability |
| Return on capital | Net profit ÷ Capital employed × 100 | Capital productivity |
| Current ratio | Current assets ÷ Current liabilities | Short-term solvency |
| Debtor days | (Debtors ÷ Revenue) × 365 | Speed of cash collection |
| Creditor days | (Creditors ÷ COGS) × 365 | Supplier payment timing |
| Stock days | (Stock ÷ COGS) × 365 | Inventory efficiency |

### Non-Financial Performance Measures

Financial measures are lagging indicators — they show what happened. Non-financial measures often signal what is coming:

| Category | Measure | Leading Indicator For |
|---|---|---|
| **Customer** | Repeat purchase rate | Revenue sustainability |
| **Customer** | Complaint rate | Future churn |
| **Operations** | Capacity utilisation % | Fixed cost absorption |
| **Operations** | Defect/reject rate | Rework costs and reputation |
| **People** | Staff turnover | Training costs, service quality |
| **People** | Days lost to illness | Productivity and morale |

### Benchmarking

Compare your performance against relevant external references:
- **Competitor benchmarking** — published accounts, industry reports, supplier intelligence
- **Industry benchmarking** — trade association data, sector average ratios
- **Best-in-class benchmarking** — learn from the best operator in your sector, regardless of geography

> **Business plan implication:** Include benchmark ratios when claiming your projections are realistic. "Industry gross margins for food retail in Uganda range from 18–25%; our projection of 22% is within this range" is far more credible than an unanchored number.

### Stakeholder Objectives in Performance Measurement

Different stakeholders measure different things:

| Stakeholder | Primary Concern |
|---|---|
| Investors/owners | Return on investment, growth |
| Lenders/banks | Cash flow coverage, debt repayment |
| Customers | Quality, price, availability |
| Employees | Job security, pay, working conditions |
| Government/URA | Tax compliance, employment |
| Community | Environmental impact, local employment |

Business plans should demonstrate how the financials serve the primary stakeholders — usually investors and lenders.

## 4. Forecasting Principles (Morlidge)

### Forecast ≠ Prediction ≠ Target ≠ Plan

These four concepts are often confused, with damaging consequences:

| Term | Definition | Purpose |
|---|---|---|
| **Forecast** | What you think will happen, based on stated assumptions | Inform decisions — it should change as assumptions change |
| **Prediction** | A statement of what will happen (certainty implied) | Not useful; no forecast is ever a prediction |
| **Target** | What you want to happen | Set ambition and motivate effort |
| **Plan** | The actions you will take to achieve the target | Execution roadmap |
| **Budget** | A financial expression of the plan for a period | Control and resource allocation |

**The Cassandra paradox:** A perfect prediction is perfectly useless. If you act on a forecast of problems ahead, you prevent the problems — and the forecast appears wrong. Forecasts should be acted on; the point is not accuracy, it is enabling better decisions.

**Critical distinction for business plans:** The financial projections in a business plan contain **both** targets (revenue ambitions) and forecasts (cost assumptions). Keep these separate in your assumptions document. Treat revenue projections as targets-with-a-plan; treat cost projections as engineering estimates.

### The TARAC/CARAT Quality Framework

A good forecast has five qualities (remembered as CARAT — the measure of precious materials):

| Quality | Meaning | Business Plan Implication |
|---|---|---|
| **T — Timely** | Available in time for meaningful action | Monthly projections enable monthly decisions in Year 1 |
| **A — Actionable** | Links to decisions that can be made | Forecast should show which levers to pull if performance diverges |
| **R — Reliable** | Unbiased; errors are random, not systematic | Projections should not consistently understate costs or overstate revenue |
| **A — Aligned** | Consistent across different parts of the organisation | Revenue, cost, and cash forecasts must reconcile with each other |
| **C — Cost Effective** | Value of the forecast > cost of producing it | Don't forecast 50 line items monthly if only 5 are material |

### 7 Symptoms of Forecasting Disease

Watch for these patterns when reviewing financial projections:

| Symptom | What It Looks Like | The Fix |
|---|---|---|
| **#1 Semantic schizophrenia** | Asking for a "best estimate" but then punishing people when it turns out to be wrong — so people produce what you want to see, not what they believe | Separate forecast from target; never "shoot the messenger" |
| **#2 Single point tunnel vision** | Obsessing over one forecast number; refusing to present ranges | Present scenarios (bear/base/bull) not just one number |
| **#3 Delusions of accuracy** | Believing more decimal places = more accurate; excessive precision in uncertain conditions | Match precision to the underlying uncertainty |
| **#4 Nervous system breakdown** | Going into ever more granular detail hoping to improve accuracy; "spreadsheet hell" | Focus detail on the 20% of line items driving 80% of value |
| **#5 Visual impairment** | Only forecasting to year-end; losing visibility as the year progresses | Use rolling forecasts — always maintain 12 months of forward visibility |
| **#6 Lack of coordination** | Finance, sales, and operations each have different forecasts; no single source of truth | One integrated model with locked assumptions |
| **#7 Socio-pathological behaviour** | Forecasters shade numbers to avoid conflict; management games the process | Separate the forecasting process from performance evaluation |

### Rolling Forecasts vs. Annual Budgets

| Annual Budget | Rolling Forecast |
|---|---|
| Fixed horizon (Jan–Dec) | Consistent horizon (always 12–18 months ahead) |
| Shrinking visibility as year progresses | Constant visibility — always see the same distance ahead |
| Set once a year | Updated monthly or quarterly |
| Becomes stale when circumstances change | Responds to new information |
| Useful for control and accountability | Useful for decision-making and steering |

**Practical guidance for business plans:** For Year 1, present monthly projections (12-month rolling view). For Years 2–3, present quarterly. Show a clear review cadence — monthly for the first year, quarterly thereafter. This demonstrates forecasting discipline and gives investors confidence the business will be managed adaptively.

### Driver-Based Forecasting

Rather than forecasting every line item independently, build the model around a small number of **key drivers** — the variables that determine everything else:

**Example: Retail business drivers**
```
Key Driver: Customer transactions per month
    → Revenue = Transactions × Average basket value
    → COGS = Revenue × COGS%
    → Staff cost = Transactions ÷ Transactions per staff-hour × Staff hourly rate
    → Space cost = fixed (rent)
    → Net profit = Revenue − COGS − Staff − Space − Other overheads
```

This approach:
- Makes assumptions explicit and auditable
- Enables instant scenario modelling (change one driver, see all impacts)
- Reduces the chance of inconsistencies across statements
- Is far faster to update than line-by-line forecasting

**Driver-based model construction:** Identify 5–10 drivers that explain 80% of the business's financials. Model everything else as a function of those drivers.

## 5. Demand vs. Sales — A Critical Distinction (Vandeput)

**Demand** = what customers want, unconstrained by supply availability.

**Sales** = what customers actually bought (demand that met available supply).

For forecasting purposes:
- **Forecast demand**, not sales — if you were out of stock last month, your sales understate your true demand
- **Sales history understates demand** when stock-outs, rationing, or supply constraints occurred
- **Adjust historical sales upward** for known stock-out periods before using as a forecast base

**Practical implications for business plans:**
- If you are entering a market where the competitor was frequently out of stock, your true addressable demand may be higher than their historical sales suggest
- Revenue projections based purely on extrapolating the previous operator's sales will be conservative if they were supply-constrained
- Build in a demand assumption that reflects your planned supply availability

### Simple Forecasting Benchmarks (Vandeput)

For businesses without sophisticated forecasting systems, use these simple benchmarks to validate revenue projections:

| Benchmark | How to Apply |
|---|---|
| **Naïve forecast** | "Next month = this month." Baseline — your forecast should beat this |
| **Moving average (3–6 periods)** | Average of last 3–6 months. Simple but effective; usually beats naïve by 10–30% |
| **Seasonal moving average** | Same month last year, adjusted for trend. Use for businesses with predictable seasonality (schools, agriculture, retail peaks) |

If your revenue projections cannot beat a simple 3-month moving average on historical data, the growth assumptions need more justification.

### Judgmental Forecasting Biases (Vandeput)

When projections are built on human judgement rather than data, three bias types emerge:

| Bias Type | Example | Counter-measure |
|---|---|---|
| **Cognitive bias** | Anchoring to first number discussed; optimism about own products | Use multiple base assumptions; compare to industry benchmarks |
| **Motivational bias** | Salesperson inflates forecast to appear ambitious; founder inflates to secure funding | Separate forecasting process from incentive assessment |
| **Process bias** | Consensus pressure in group forecasting; senior person's estimate adopted without challenge | Anonymous submission of independent estimates; structured review |

> **For business plans:** Revenue projections are the most bias-prone part. The antidote is bottom-up forecasting: build from known customer acquisition rates, conversion data, market size constraints, and unit economics — not from a target revenue figure worked backwards.

## 6. Cash Budget Template (ILM Framework)

The cash budget is the most important budget for new and small businesses. Format:

| | Month 1 | Month 2 | Month 3 | ... |
|---|---|---|---|---|
| **Opening cash balance** | — | (M1 closing) | (M2 closing) | |
| **Cash inflows** | | | | |
| Sales receipts (timing-adjusted) | | | | |
| Loans received | | | | |
| Other income | | | | |
| **Total inflows** | | | | |
| **Cash outflows** | | | | |
| Stock/raw material purchases | | | | |
| Staff wages | | | | |
| Rent | | | | |
| Utilities | | | | |
| Loan repayments | | | | |
| CapEx payments | | | | |
| Tax payments (quarterly/annual) | | | | |
| **Total outflows** | | | | |
| **Net cash movement** | | | | |
| **Closing cash balance** | | | | |

**Key disciplines:**
- Cash is received when the customer pays, not when the sale is made
- Cash is paid when you pay the supplier, not when you receive stock
- Show the timing of NSSF, PAYE, and URA payments (quarterly or monthly)
- Highlight the lowest monthly balance — this is the peak funding requirement
