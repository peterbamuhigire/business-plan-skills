## Financial Ratio Analysis

### Key Ratio Categories

#### 1. Profitability Ratios

```
Return on Sales (ROS) = Net Income After Tax / Net Sales
EAF&G: UGX 729M / UGX 49.7B = 1.47%

Return on Assets (ROA) = Net Income After Tax / Total Assets
EAF&G: UGX 729M / UGX 24.8B = 2.95%

Return on Equity (ROE) = Net Income After Tax / Owner's Equity
EAF&G: UGX 729M / UGX 13.6B = 5.37%
```

#### 2. Liquidity Ratios

```
Current Ratio = Current Assets / Current Liabilities
EAF&G: UGX 15.9B / UGX 6.9B = 2.30

Quick Ratio = (Current Assets - Inventory) / Current Liabilities
EAF&G: (UGX 15.9B - UGX 9.3B) / UGX 6.9B = 0.96
```

**Guideline:** Current ratio of 2.0 or higher is generally considered healthy. Quick ratio below 1.0 indicates potential liquidity risk if inventory cannot be sold quickly.

#### 3. Solvency Ratios

```
Debt-to-Equity = Total Liabilities / Owner's Equity
EAF&G: UGX 11.2B / UGX 13.6B = 0.82

Equity-to-Asset = Owner's Equity / Total Assets
EAF&G: UGX 13.6B / UGX 24.8B = 54.9%
```

#### 4. Efficiency Ratios

```
Asset Turnover = Net Sales / Total Assets
EAF&G: UGX 49.7B / UGX 24.8B = 2.01

Inventory Turnover = Cost of Goods Sold / Average Inventory
EAF&G: UGX 43.5B / UGX 9.3B = 4.69

Accounts Receivable Turnover = Net Sales / Accounts Receivable
EAF&G: UGX 49.7B / UGX 6B = 8.38

Wage Efficiency Ratio = Labour Cost / Net Sales
EAF&G: UGX 1.6B / UGX 49.7B = 3.3%
```

### The Profitability Analysis Model (DuPont Model)

The DuPont model breaks ROE into three "paths to profit":

```
ROE = Return on Sales x Asset Turnover x Leverage Ratio

ROE = (Net Income / Net Sales) x (Net Sales / Total Assets) x (Total Assets / Owner's Equity)

EAF&G: 1.47% x 2.01 x 1.82 = 5.37%
```

| Path | Ratio | EAF&G Value | Manager Focus |
|------|-------|-----------|---------------|
| Operating path | Return on Sales | 1.47% | Sales revenue, cost control, pricing |
| Asset efficiency path | Asset Turnover | 2.01 | Inventory management, asset utilisation |
| Debt path | Leverage Ratio | 1.82 | Optimal use of borrowed funds |

**Key insight:** Improvements in any path improve ROE, but changes are interrelated. Using more debt raises leverage (increasing ROE) but also increases interest expense (decreasing ROS).

## Volume-Cost (Break-Even) Analysis

### Cost Classification

| Cost Type | Definition | Examples |
|-----------|-----------|---------|
| **Fixed costs** | Do not change with volume of business | Depreciation, rent, insurance, property taxes, management salaries |
| **Variable costs** | Change directly with volume | Cost of goods sold, sales commissions, part-time labour, delivery fuel |
| **Semi-variable** | Have both fixed and variable components | Utilities, maintenance, telephone |

### Break-Even Calculation

```
Contribution to Overhead (CTO) = Selling Price - Variable Costs (as % of sales)
CTO = 1.00 - 0.8964 = 0.1036 (10.36% of each sales shilling)

Break-Even Sales = Fixed Costs / CTO
EAF&G: UGX 4.3B / 0.1036 = UGX 41.1B
```

### Applied Break-Even Scenarios

#### Profit Planning
```
Additional Sales for Profit Goal = Profit Goal / CTO
EAF&G target profit of UGX 925M:
UGX 925M / 0.1036 = UGX 8.9B additional sales needed

Total Sales Goal = UGX 41.1B + UGX 8.9B = UGX 50.1B
```

#### Impact of Cost Changes
```
Adding a truck (new fixed cost = UGX 37M/year):
Additional sales to cover = UGX 37M / 0.1036 = UGX 357M

Variable costs increase 1 percentage point (CTO falls to 0.0936):
New break-even = UGX 4.3B / 0.0936 = UGX 45.5B
Increase in break-even = UGX 4.4B

Variable costs decrease 1 percentage point (CTO rises to 0.1136):
New break-even = UGX 4.3B / 0.1136 = UGX 37.5B
Decrease in break-even = UGX 3.6B
```

#### Impact of Price Changes
```
2% price reduction (new CTO = 0.98 - 0.8964 = 0.0836):
New break-even = UGX 4.3B / 0.0836 = UGX 50.9B
Increase in break-even = UGX 9.8B (roughly 24%)
```

**Critical rule:** Never sell below variable cost except for loss-leader pricing or clearing obsolete inventory.

## Capital Investment Analysis

### Four Evaluation Methods

#### 1. Payback Period
```
Payback Period = Initial Investment / Annual Net Cash Inflow
```
Simple but ignores time value of money and cash flows beyond payback.

#### 2. Simple Rate of Return
```
Simple Rate of Return = Average Annual Net Income / Initial Investment x 100

Example — Self-propelled fertiliser applicator:
Purchase price: UGX 520M | Salvage value: UGX 93M
Average annual net income: UGX 61M
Simple Rate of Return = UGX 61M / UGX 520M = 11.7%
```

#### 3. Net Present Value (NPV)

```
NPV = Sum of [Net Cash Inflow / (1 + i)^n] - Initial Investment

Decision rule:
  NPV > 0  -->  Accept (earns more than discount rate)
  NPV = 0  -->  Indifferent
  NPV < 0  -->  Reject (earns less than discount rate)
```

**Present Value Factors (selected):**

| Year | 5% | 8% | 10% | 12% | 14% |
|------|------|------|-------|-------|-------|
| 1 | 0.9524 | 0.9259 | 0.9091 | 0.8929 | 0.8772 |
| 3 | 0.8638 | 0.7938 | 0.7513 | 0.7118 | 0.6750 |
| 5 | 0.7835 | 0.6806 | 0.6209 | 0.5674 | 0.5194 |
| 10 | 0.6139 | 0.4632 | 0.3855 | 0.3223 | 0.2697 |
| 20 | 0.3769 | 0.2145 | 0.1486 | 0.1037 | 0.0728 |

#### 4. Internal Rate of Return (IRR)

The discount rate at which NPV = 0. Found by interpolation between two discount rates:

```
If NPV at 14% = UGX 13.1M and NPV at 15% = -UGX 3.7M:
IRR = 14% + [UGX 13.1M / (UGX 13.1M + UGX 3.7M)] x 1% = 14.78%

Decision rule: Accept if IRR > required rate of return (cost of capital)
```

### Compounding and Discounting

```
Future Value = Present Value x (1 + i)^n
Present Value = Future Value / (1 + i)^n

Example: UGX 3.7M invested at 5% for 4 years:
FV = UGX 3.7M x (1.05)^4 = UGX 3.7M x 1.2155 = UGX 4.5M

Present value of UGX 3.7M received in 4 years at 5%:
PV = UGX 3.7M x 0.8227 = UGX 3.04M
```

## Financing the Agribusiness

### Types of Capital

| Type | Duration | Purpose | Collateral |
|------|----------|---------|-----------|
| **Short-term loans** | Up to 1 year | Seasonal inventory, accounts receivable | Current assets, personal guarantee |
| **Intermediate-term** | 1-5 years | Equipment, working capital growth | Fixed assets, equipment |
| **Long-term loans** | Over 5 years | Land, buildings, major facilities | Mortgage on fixed assets |
| **Equity capital** | No due date | Permanent capital base | No repayment required |

### External Financing Sources

- **Commercial banks** — Most common source; offer lines of credit, term loans
- **Agricultural credit institutions** — Uganda Development Bank, agricultural development funds
- **Government programmes** — Small business loans, rural development grants, Youth Livelihood Programme
- **Leasing** — Operating leases (short-term, off-balance-sheet) and capital leases (long-term, on-balance-sheet)
- **Bonds and debentures** — For larger firms; convertible bonds can later convert to equity
- **Equity markets** — Common stock, preferred stock (6% typical dividend), retained earnings

## Operations Management

### Production Planning

Operations management covers the direction and control of processes used to produce goods and services. Key decision areas:

#### Facility Location Factors

| Factor | Considerations |
|--------|---------------|
| **Proximity to raw materials** | Reduces transport costs; critical for perishable inputs |
| **Proximity to markets** | Reduces delivery time and cost; important for perishable outputs |
| **Labour availability and cost** | Seasonal labour needs; skill levels required |
| **Transport infrastructure** | Road, rail, port access; critical for bulk commodities |
| **Utilities and services** | Water supply, energy, waste disposal |
| **Regulatory environment** | Zoning, environmental permits, tax incentives |
| **Community factors** | Quality of life, housing, schools (for attracting staff) |

#### Facility Layout Types

| Layout | Description | Example |
|--------|-------------|---------|
| **Process layout** | Groups similar equipment/activities together | Farm implement manufacturing |
| **Product layout** | Arranges equipment in sequence of production steps | Fluid milk processing plant |
| **Hybrid layout** | Combines process and product approaches | Supermarket (departments + checkout flow) |
| **Fixed-position** | Product stays in place; workers and equipment come to it | Farm building construction |

#### Job Design Objectives

1. Meet production and quality goals of the organisation
2. Make jobs safe, satisfying, and motivating for employees

Modern approaches move beyond Taylorist scientific management to include job enlargement (broader task range), job enrichment (greater autonomy and responsibility), and self-directed work teams.

### Unique Characteristics of Agricultural Operations

- **Seasonality** — Harvest periods create extreme demand peaks; capacity must handle peak loads
- **Perishability** — Time-critical processing; cold chain requirements
- **Biological variability** — Input quality varies by season, weather, and growing conditions
- **Bulk handling** — Large volumes of low-value commodities require efficient logistics
- **Food safety** — Strict regulatory requirements (HACCP, traceability)
- **Weather dependence** — Production schedules subject to uncontrollable weather events

## Supply Chain Management

### The Supply Chain Process

Supply chain management integrates procurement, production planning, inventory management, and distribution into a coordinated system.

#### Key Components

1. **Master Production Schedule (MPS)** — Plans what to produce, in what quantities, and when
2. **Materials Requirements Planning (MRP)** — Determines what materials are needed and when to order them
3. **Inventory Management** — Balances carrying costs against stockout risks
4. **Distribution Management** — Warehousing, transportation, and carrier selection

### Inventory Management

#### Reasons for Holding Inventory

| Reason | Example |
|--------|---------|
| **Buffer against uncertainty** | Unpredictable demand, supplier delays |
| **Seasonal production** | Harvest-time grain storage for year-round sale |
| **Quantity discounts** | Buying fertiliser in bulk at lower unit cost |
| **Pipeline stock** | Goods in transit between locations |
| **Speculation** | Holding grain waiting for higher prices |

#### Inventory Review Systems

| System | How It Works | Best For |
|--------|-------------|----------|
| **Periodic review** | Check inventory at fixed intervals; order to target level | Lower-value items, stable demand |
| **Continuous review** | Monitor continuously; order fixed quantity when reorder point reached | High-value items, variable demand |

### Distribution Components

**Warehousing options:**
- **Forward placement** — Stock near customers for fast delivery; higher warehousing cost, lower transport cost
- **Backward placement** — Centralised storage near production; lower warehousing cost, higher transport cost

**Transportation modes for agriculture:**

| Mode | Advantages | Disadvantages | Common Uses |
|------|-----------|--------------|-------------|
| **Truck** | Flexible, door-to-door | Higher cost per ton-mile | Short-haul, final delivery |
| **Rail** | Low cost for bulk | Limited flexibility, slower | Grain, fertiliser, bulk commodities |
| **Barge/ship** | Lowest cost per ton-mile | Very slow, limited routes | Export grain, bulk commodities |
| **Container** | Intermodal flexibility | Requires infrastructure | Processed foods, exports |

### Shift to Supply Chain Management

Traditional procurement focused on lowest purchase price. Modern supply chain management emphasises:

- Total cost of ownership (price + transport + handling + quality + reliability)
- Fewer, deeper supplier relationships (partnership model)
- Information sharing across the chain
- Just-in-time delivery to reduce inventory costs
- Quality at the source (supplier responsibility)

## Human Resource Management

### Organisational Structures

| Structure | Characteristics | Best For |
|-----------|----------------|----------|
| **Line organisation** | Direct chain of command; simple hierarchy | Small agribusinesses |
| **Line and staff** | Line authority plus specialist advisors | Medium-sized firms |
| **Functional** | Organised by specialist functions | Larger firms needing expertise |

### HR Management Process

1. **Job analysis** — Determine duties, responsibilities, and required qualifications
2. **Job description** — Written document specifying tasks, authority, and evaluation criteria
3. **Job specification** — Minimum qualifications (education, experience, skills)
4. **Recruitment** — Internal promotion, external advertising, referrals
5. **Selection** — Application review, interviews, reference checks
6. **Training and development** — Orientation, on-the-job training, continuing education
7. **Performance evaluation** — Regular assessment against predetermined standards
8. **Compensation** — Wages, benefits, incentives aligned with performance

### Leadership Styles

| Style | Manager Role | Employee Role | Best When |
|-------|-------------|---------------|-----------|
| **Autocratic** | Tells, decides alone | Follows instructions | Crisis situations, unskilled workers |
| **Democratic** | Consults, shares decision-making | Participates, contributes ideas | Experienced teams, complex problems |
| **Laissez-faire** | Delegates, minimal direction | Self-directed, autonomous | Highly skilled professional teams |
| **Transformational** | Inspires, creates vision | Motivated by purpose and growth | Change management, innovation |

### Motivation Theories Applied to Agribusiness

**Maslow's Hierarchy of Needs:**
1. Physiological (fair wages, safe working conditions)
2. Safety (job security, benefits, stable employment)
3. Social (team belonging, workplace relationships)
4. Esteem (recognition, responsibility, advancement)
5. Self-actualisation (meaningful work, professional growth)

**Herzberg's Two-Factor Theory:**

| Hygiene Factors (prevent dissatisfaction) | Motivators (create satisfaction) |
|------------------------------------------|--------------------------------|
| Pay and benefits | Achievement and recognition |
| Working conditions | Responsibility and autonomy |
| Company policies | Opportunity for advancement |
| Job security | Meaningful, challenging work |
| Supervision quality | Professional growth |

**Key insight for agribusiness:** Seasonal workers and entry-level positions require attention to hygiene factors first. Retaining skilled managers and technical staff requires emphasis on motivators — recognition, autonomy, and growth opportunities.
