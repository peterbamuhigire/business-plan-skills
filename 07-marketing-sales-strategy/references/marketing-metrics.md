# Marketing Metrics Reference

Comprehensive marketing measurement formulas from Farris's "Marketing Metrics" for tracking, evaluating, and optimising marketing performance.

## 1. Campaign and Channel Metrics

### Cost Per Lead (CPL)

```
CPL = Total channel spend / Number of leads generated
```

### Cost Per Acquisition (CPA) / Customer Acquisition Cost (CAC)

```
CPA = Total marketing and sales spend / New customers acquired

Channel CPA = Channel spend / Customers from that channel
```

### Conversion Rate

```
Conversion Rate = Conversions / Total visitors or leads x 100
```

### Funnel Conversion Rates

```
Lead-to-MQL = Marketing Qualified Leads / Total leads x 100
MQL-to-SQL = Sales Qualified Leads / MQLs x 100
SQL-to-Opportunity = Opportunities / SQLs x 100
Opportunity-to-Close = Closed deals / Opportunities x 100

Overall conversion = Product of all stage rates
```

### Click-Through Rate (CTR)

```
CTR = Clicks / Impressions x 100
```

| Channel | Good CTR | Great CTR |
|---|---|---|
| Search ads | 3-5% | > 5% |
| Display ads | 0.3-0.5% | > 0.5% |
| Email | 2-3% | > 3% |
| Social ads | 0.5-1.5% | > 1.5% |

### Cost Per Click (CPC)

```
CPC = Total ad spend / Total clicks
```

### Cost Per Mille (CPM)

```
CPM = (Total ad spend / Total impressions) x 1,000
```

## 2. Marketing ROI and Efficiency

### Marketing ROI (Farris)

```
Marketing ROI = (Incremental revenue from marketing - Marketing cost) / Marketing cost x 100
```

**Conservative version:**
```
Marketing ROI = (Incremental margin from marketing - Marketing cost) / Marketing cost x 100
```

### Return on Ad Spend (ROAS)

```
ROAS = Revenue from ads / Ad spend

Minimum viable ROAS = 1 / Gross margin %
Example: If margin is 50%, minimum ROAS = 2.0 (break even)
```

| ROAS | Assessment |
|---|---|
| < 1.0 | Losing money on ads |
| 1.0 - 2.0 | Below break-even for most businesses |
| 2.0 - 4.0 | Acceptable (depends on margin) |
| 4.0 - 8.0 | Good performance |
| > 8.0 | Excellent — consider scaling |

### Marketing Efficiency Ratio

```
MER = Total revenue / Total marketing spend
```

Unlike ROAS (which is per-channel), MER measures overall marketing efficiency including organic and brand-driven revenue.

### Customer Payback Period

```
Payback months = CAC / (Monthly ARPU x Gross margin %)
```

## 3. Brand and Awareness Metrics

### Share of Voice (SOV)

```
SOV = Brand's advertising spend / Total category advertising spend x 100
```

**SOV-SOM relationship (Farris):**
- SOV > SOM → Market share tends to grow (excess share of voice)
- SOV = SOM → Market share tends to stay stable
- SOV < SOM → Market share tends to decline

### Brand Awareness

```
Aided awareness = % who recognise brand when prompted
Unaided awareness = % who name brand without prompting (top of mind)
```

### Net Promoter Score (NPS)

```
NPS = % Promoters (9-10) - % Detractors (0-6)
```

| NPS | Assessment |
|---|---|
| < 0 | Serious problems |
| 0-30 | Good |
| 30-50 | Very good |
| 50-70 | Excellent |
| > 70 | World class |

### Customer Satisfaction (CSAT)

```
CSAT = Satisfied responses / Total responses x 100
```

## 4. Digital Marketing Metrics

### Email Marketing

```
Open rate = Emails opened / Emails delivered x 100
Click rate = Links clicked / Emails delivered x 100
Unsubscribe rate = Unsubscribes / Emails delivered x 100
List growth rate = (New subscribers - Unsubscribes) / Total subscribers x 100
```

| Metric | Good | Excellent |
|---|---|---|
| Open rate | 20-25% | > 30% |
| Click rate | 2-3% | > 5% |
| Unsubscribe rate | < 0.5% | < 0.2% |

### Social Media

```
Engagement rate = (Likes + Comments + Shares) / Followers x 100
Reach rate = Unique impressions / Followers x 100
Social conversion rate = Conversions from social / Total social clicks x 100
```

### SEO / Content

```
Organic traffic growth = (Current organic - Prior organic) / Prior organic x 100
Domain authority = Third-party score (Moz, Ahrefs) out of 100
Keyword rankings = Number of target keywords in top 10
Content conversion rate = Conversions from content / Content page visits x 100
```

### Attribution Models

| Model | How It Works | Best For |
|---|---|---|
| **Last click** | 100% credit to last touchpoint | Simple, direct-response |
| **First click** | 100% credit to first touchpoint | Understanding discovery |
| **Linear** | Equal credit to all touchpoints | Balanced view |
| **Time decay** | More credit to recent touches | Long sales cycles |
| **Position-based** | 40/20/40 first/middle/last | Balanced with emphasis on discovery and conversion |

## 5. Pricing Metrics

### Price Elasticity

```
Price elasticity = % change in quantity demanded / % change in price
```

| Elasticity | Meaning |
|---|---|
| > 1 (elastic) | Demand sensitive to price — small increase = large demand drop |
| = 1 (unit elastic) | Proportional response |
| < 1 (inelastic) | Demand insensitive — can raise prices with limited impact |

### Margin Analysis

```
Gross margin = (Revenue - COGS) / Revenue x 100
Contribution margin = (Revenue - Variable costs) / Revenue x 100
Break-even volume = Fixed costs / Contribution margin per unit
```

### Discount Impact

```
Volume increase needed = Discount % / (Margin % - Discount %)

Example: 20% discount on 50% margin product
Volume increase needed = 20 / (50 - 20) = 67% more volume to break even
```

This formula reveals why discounting is dangerous for low-margin businesses.
