# World-Class Consultancy Engine Roadmap

Date: 2026-04-14

## Purpose

This roadmap converts the current repository from a strong business-plan skills suite into a full advisory production system that can generate:

- bankable business plans
- investor-grade business plans
- project proposals
- business proposals
- grant proposals
- pitch materials
- execution roadmaps
- strategy-development outputs that read like top-tier consulting work

The plan is based on:

- the repository's current structure and standards in `AGENTS.md`
- the latest repo evaluation in `docs/evaluation/2026-04-12/`
- the March completion blueprints in `docs/march-18-analysis/`
- the external analysis pack in `C:\Users\Peter\.claude\skills\docs\analysis`

## Critical Interpretation

The external analysis is valuable but only partially aligned to this repository. Its strongest contribution is operating discipline:

- close structural gaps before polishing outputs
- treat orchestration as a first-class system
- use explicit quality gates
- deepen evidence, financial rigour, and implementation realism
- build a business engine, not just a library of isolated prompts

Its weakest contribution for this repo is emphasis on software-engineering skills that are not the direct bottleneck here. For this project, the real competitive gap is not Kubernetes or Stripe. It is:

- end-to-end orchestration
- evidence discipline
- consulting-grade synthesis
- repeatable proposal production
- stronger financial and valuation logic
- local-country depth beyond Uganda
- productised delivery assets that make the system usable by a consultancy team

## End State

At the end of these phases, the repository should function as a consultancy operating system with five flagship output lines:

1. Bank loan and DFI business plans
2. Equity and investor plans
3. Client-facing business proposals
4. Donor and grant proposals
5. Strategy-development and execution programmes

## Competitive Ambition

The practical goal is not to imitate McKinsey, Bain, EY, or the Big Four in every respect. The goal is to outperform them in the segments where this system can realistically win:

- East Africa and frontier-market context
- SME and mid-market bankability work
- grant and development-finance preparation
- speed from intake to high-quality first draft
- tighter integration between strategy, financials, and implementation
- lower delivery cost with high analytical discipline

That is a credible path to beating larger firms on relevance, speed, and value density, even if they still retain scale advantages in proprietary databases and large analyst teams.

## Sequence Logic

- Phase 01 defines what "world-class" means commercially and operationally.
- Phase 02 makes the system controllable end to end.
- Phase 03 improves evidence quality so later outputs do not rest on weak claims.
- Phase 04 upgrades the plan-writing core.
- Phase 05 strengthens the numerical credibility that determines lender and investor trust.
- Phase 06 broadens the engine into proposals and grants.
- Phase 07 enforces advisory-grade synthesis and review discipline.
- Phase 08 makes the system transferable, repeatable, and saleable as a consultancy engine.
- Phase 09 builds moat, reputation, and feedback loops so quality compounds.

## Completion Standard

This roadmap is complete only when the repository can consistently produce outputs that are:

- conclusion-led
- evidence-backed
- financially coherent
- audience-specific
- cross-section consistent
- locally grounded
- implementation-ready
- deliverable by a trained team, not only by the founder
