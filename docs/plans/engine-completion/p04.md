# Phase 04 - Core Business Plan System Upgrade

## Objective

Upgrade the numbered plan-section skills so the output is not just complete, but structurally superior: thesis-led, evidence-backed, internally consistent, and audience-aware.

## Why This Phase Follows Research and Orchestration

The numbered sections are the heart of the repository. They should be upgraded only after:

- the operating model is clear
- orchestration exists
- evidence standards are stronger

Otherwise you risk rewriting sections against unstable requirements.

## Core Workstreams

### 1. Enforce conclusion-first drafting

Across the core plan sections, add stronger guidance for:

- assertion-evidence-implication structure
- hypothesis-led section openings
- conclusion-first headings
- "so what" discipline

### 2. Upgrade strategic sections

Deepen:

- `02-company-overview`
- `03-products-services`
- `06-competitive-analysis`
- `08-operations-plan`
- `09-management-team`

Key additions should include:

- governance and ownership logic
- board and advisory architecture
- moat and strategic-position clarity
- operating-capability realism
- founder-gap mitigation

### 3. Improve implementation realism

Upgrade:

- `13-implementation-timeline`
- `16-sustainability-strategy`
- `15-appendices`

So they support:

- first-90-days mobilisation
- implementation dependencies
- sustainability and ESMP expectations
- evidence packaging by audience type

### 4. Add output-type variants

Core plan skills should adapt based on whether the output is for:

- bank debt
- equity capital
- DFI or impact capital
- internal strategy
- strategic partner discussions

## Repository Changes to Make

- systematic pass through core numbered skills
- new cross-references to audience modes, evidence matrix, and consistency gates
- stronger appendix packaging standards

## Deliverables

- upgraded core section skills
- audience-mode variants embedded in workflow
- stronger governance, implementation, and appendix logic

## Exit Criteria

Phase 04 is complete only when:

- the full plan reads like one investment or strategy case
- section openings state conclusions, not only topics
- governance, operations, and implementation are decision-grade rather than descriptive
- appendix standards are strong enough for diligence-heavy engagements

## Risks if Skipped

- the system remains technically complete but commercially average
- outputs read like stitched sections instead of advisory-grade documents
- stronger evidence still gets diluted by weak narrative structure

## Suggested Reading Material

- *Bankable Business Plans* by Edward Rogoff
- *On Target* by Tim Berry and Doug Wilson
- *The Portable MBA in Strategy* edited by Liam Fahey and Robert Randall
- *Understanding Michael Porter* by Joan Magretta

Buy and provide these now because this phase is about sharpening the actual section logic and strategic quality of the business-plan core.
