## Section-by-Section Assessment

Each section is assessed on four criteria:
- **Content completeness** — Does the skill produce all required content for that section?
- **Framework depth** — Does it apply named, credible methodologies?
- **Uganda localisation** — Is it calibrated to the EA context?
- **Cross-referencing** — Does it connect to adjacent skills and reference files?

### 06 — Competitive Analysis
**File:** `06-competitive-analysis/SKILL.md`
**Status: ADEQUATE — needs strengthening ✅⚠️**

| Criterion | Score | Notes |
|-----------|-------|-------|
| Content completeness | 8/10 | SWOT, competitive matrix, moat types, positioning |
| Framework depth | 7/10 | Teece business model vs strategy, Kaza differentiation — but Porter's Five Forces MISSING |
| Uganda localisation | 6/10 | Generic; no Uganda-specific competitive dynamics examples |
| Cross-referencing | 7/10 | References teece and kaza files; OSINT from meta-due-diligence not wired |

**This is the weakest of the core 15 skills relative to world-class standards.** Porter's Five Forces is conspicuously absent. The OSINT competitor intelligence methodology in `meta-due-diligence/references/osint-business-intelligence.md` is not referenced here, which means competitor intelligence and competitive analysis are not integrated.

**Recommended additions:**
- Porter's Five Forces framework with Uganda/EA application examples
- PESTLE framework (even a brief version) as a named tool
- Cross-reference to `meta-due-diligence/references/osint-business-intelligence.md` for competitor intelligence gathering
- Morningstar's five moat types (network effects, switching costs, cost advantages, intangibles, efficient scale) as a scoring rubric
