## Methodology Canon Assessment

The suite's methodology canon is **stronger than most consulting firm deliverables for SME-scale businesses in Africa**. The inclusion of:
- Minto (Pyramid Principle) for document structure
- Rasiel (MECE) for analysis structure
- Damodaran (DCF/WACC) for valuation
- Klaff (STRONG method) for pitching
- Duarte (Sparkline, assertion-evidence) for presentation design
- Feld/Mendelson (term sheets) for equity deals
- Howson + Scharfman + Berkman (DD) for investor readiness

...represents a canon that a McKinsey associate would recognise and respect.

The gap is Porter, which is so fundamental that its absence is a credibility signal. Fix it with Task 1.3 and the canon is complete.
